# Responsive Personal Portfolio Website

This project is a **responsive personal portfolio webpage** developed as part of an **internship task at Swizosoft Private Limited**.

## 📌 Task Description
Create a responsive personal portfolio webpage using HTML and CSS concepts, including:
- HTML structure (header, sections, footer)
- Tables, lists, media elements
- CSS colors, borders, box model
- Flexbox for layout alignment
- Media queries for mobile responsiveness

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## 📄 Features
- Header with navigation menu
- Hero section with name and introduction
- About section
- Skills section (table format)
- Projects section
- Contact section
- Footer
- Fully responsive for mobile view

## 📱 Responsiveness
The website is designed to adapt to different screen sizes using CSS media queries, ensuring a clean and user-friendly experience on both desktop and mobile devices.

## 🚀 How to Run
1. Clone the repository or download the project files
2. Open `ac.html` in any web browser
